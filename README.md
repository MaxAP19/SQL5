# SQL5
A virtual view and physical snapshot of data.

## Описание:
Проект посвящён работе с виртуальными таблицами (VIEW) и материализованными представлениями (MATERIALIZED VIEW) в PostgreSQL. Основной фокус — абстракция данных, повторное использование запросов и управление актуальностью данных.

## Этапы выполнения проекта:
- Создание и использование представлений (VIEW) для фильтрации и объединения данных.
- Генерация и хранение производных данных (дат) во View.
- Поиск пропущенных данных и выполнение операций над множествами с сохранением результата во View.
- Расчёт производных показателей (скидочная цена) в представлениях.
- Создание, обновление и удаление материализованных представлений (MATERIALIZED VIEW).

## Освоенные компетенции:
- Понимание различий между VIEW и MATERIALIZED VIEW и сценариев их применения.
- Навыки проектирования виртуальных слоёв доступа к данным.
- Работа с генерацией данных и операциями теории множеств в SQL.
- Управление актуальностью данных через обновление материализованных представлений.
- Повышение читаемости и переиспользуемости SQL-решений.
